Install the `bindfinder-key-packs` skill from this repository.

If your agent supports `$skill-installer`, run:

```text
Use $skill-installer to install https://github.com/younesehb/bindfinder/tree/main/skills/bindfinder-key-packs
```

After installation, restart Codex so the new skill is available.

Then use it like:

```text
$bindfinder-key-packs I use Vim. Find my config, extract my custom keybindings, and add them to bindfinder entries.
```
