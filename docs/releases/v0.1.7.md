# v0.1.7

This release improves the post-install guidance.

Highlights:

- installer output now shows the default shortcut for the detected setup
- installer output now prints the exact reload command for bash, zsh, or fish when needed
- post-install guidance stays short and points users to `bindfinder --help` and the docs

Assets:

- `install.sh`
- `bindfinder-0.1.7-x86_64-unknown-linux-musl.tar.gz`
- `bindfinder-0.1.7-aarch64-apple-darwin.tar.gz`
