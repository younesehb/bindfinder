# v0.1.1

This release turns the initial prototype into a usable tool for daily terminal work.

Highlights:

- search-first TUI flow with normal/search modes
- tmux integration that opens the picker and pastes the selected command back into the original pane
- favorites, hidden entries, hidden tools, and favorites-only filtering
- YAML-configurable in-app keybindings, including multi-step bindings like `g g`
- navi repository import support
- shipped man page with `bindfinder install man --write`
- Linux/macOS path handling, Homebrew formula, and GitHub Actions CI/release workflows

Assets:

- `bindfinder-0.1.1-x86_64-unknown-linux-gnu.tar.gz`
- `bindfinder-0.1.1-aarch64-apple-darwin.tar.gz`
