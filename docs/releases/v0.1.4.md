# v0.1.4

This release fixes Linux installation portability.

Highlights:

- Linux release artifacts now target `x86_64-unknown-linux-musl` instead of glibc-linked `gnu`
- the installer validates the downloaded binary before running setup
- Linux install docs now describe the musl-based artifact correctly

Assets:

- `bindfinder-0.1.4-x86_64-unknown-linux-musl.tar.gz`
- `bindfinder-0.1.4-aarch64-apple-darwin.tar.gz`
