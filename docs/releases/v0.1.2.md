# v0.1.2

This release focuses on making the shell flow feel natural.

Highlights:

- commands with placeholders now open an argument form inside the TUI before insertion
- bare `bindfinder` in an interactive shell now uses the shell-integrated picker path
- `bindfinder --help`, `bindfinder doctor`, and other subcommands still go to the real CLI
- `bf` is available as a short alias for the shell picker path
- shell integration and release checks were cleaned up and tightened

Assets:

- `bindfinder-0.1.2-x86_64-unknown-linux-gnu.tar.gz`
- `bindfinder-0.1.2-aarch64-apple-darwin.tar.gz`
