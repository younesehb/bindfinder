# v0.1.3

This release adds a clean uninstall path.

Highlights:

- `bindfinder uninstall` removes the installed binary, man page, and managed shell or tmux blocks
- `bindfinder uninstall --purge-data` also removes local config, state, packs, imported repos, and cache files
- install and uninstall docs were tightened so setup and cleanup are symmetrical

Assets:

- `bindfinder-0.1.3-x86_64-unknown-linux-gnu.tar.gz`
- `bindfinder-0.1.3-aarch64-apple-darwin.tar.gz`
