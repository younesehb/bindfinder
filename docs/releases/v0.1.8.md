# v0.1.8

This release improves the default content out of the box.

Highlights:

- installer now imports `denisidoro/cheats` automatically when `git` is available
- new installs start with a broader default command catalog instead of only built-in packs
- install docs now mention the default cheats import

Assets:

- `install.sh`
- `bindfinder-0.1.8-x86_64-unknown-linux-musl.tar.gz`
- `bindfinder-0.1.8-aarch64-apple-darwin.tar.gz`
