# v0.1.5

This release makes the install command stable.

Highlights:

- `install.sh` is now uploaded as a release asset
- README and install docs now use the stable `releases/latest/download/install.sh` URL
- installer drift between `main` and the latest tag is removed from the default install path

Assets:

- `install.sh`
- `bindfinder-0.1.5-x86_64-unknown-linux-musl.tar.gz`
- `bindfinder-0.1.5-aarch64-apple-darwin.tar.gz`
